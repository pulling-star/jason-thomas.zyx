This is a summary of my best work as a software developer and journalist.

I've never been employed as a developer, so these projects are the product of my enthusiam and natural curiosity for problem solving and language.

  • Reinforcement learing - how I taught robots the finer points of armed combat (Python)

    Github page

    This is a demonstration of machine learning, where one or more software agents learns to do its/their tasks without being explicitly programmed to do so

  • Natural language processing - looking for passive voice in the writing of George Orwell (Python, NLTK, Stanford Core NLP)

    Github page

    Before George Orwell's death, he wrote an essay recommending people use active voice, rather than passive, in their writing if they are able. So I thought it would be interesting to look at Orwell's own work for passive and actice voice.

  • Natural selection in the DOM - (JavaScript, D3.js)

    Github page Project link

    This is a data visualisation of what natural selection might look like a controlled environment. There are predators that will pick off the most-visible prey. The prey that blend are the ones that reproduce.

  • This blog - my first attempt at using a backend web framework (Python, Jinja2, Flask)

    Github page

    This was my first attempt at using Flask. This is hosted from my personal VPS. I deployed to a VPS so I could learn how Apache2, Linux and virtual hosts interact.

  • Databot - a robot/API to help users of a Slack channel find datasets (Python, Flask)

    Github page

    This is a project I contributed to - adding features and tests.

  • No More Tables generator - a way to spin up responsive tables on the go (JavaScript, CSS, HTML)

    Github page | Project link

    When tables get squishy, they look bad. This project lets you set a 'break point' for the table. If the screen size is smaller than than, the table folds in under itself. The end result uses CSS and no JavaScript to achieve this.

  • Country shape SVG exporter - a tool for artists and non-programmers to export SVG shapes of countries (JavaScript, D3.js)

    Github page | Project link

    Too often, in a news room, I've seen artists use the Mercator projection for shapes of individual countries. This globe is a much better representation of countries' actual shapes and sizes, although even a sphere has some distortion.

  • Web scrapers - varied and many (Python, Ruby, requests, BeautifulSoup4, Mechanize, PhantomJS, Selenium)

    Github page

    I wrote most of these as a journalist at SBS News. I've also written many scrapers to collect data for the website.

  • Data visualisations - varied and many (JavaScript, D3, C3, Leaflet)

    Github page

    I was employed as a data journalist at SBS News, where I created many data visualisations. These are some of my best work. The Github page has links to the dataviz.
